# CS229 - Machine Learning (Stanford) — Self Study Notes

Personal handwritten notes from self-studying Stanford's CS229 Machine Learning course by Prof. Andrew Ng.

## Why CS229?

CS229 is one of the most rigorous ML courses available. I studied it to build strong mathematical foundations in machine learning — the kind that actually helps you understand *why* algorithms work, not just how to use them.

## Key Concepts Studied

- **Linear Regression**: Normal equations, gradient descent, probabilistic interpretation (MLE)
- **Classification**: Logistic regression, decision boundaries, softmax
- **Generative Models**: Gaussian Discriminant Analysis, Naive Bayes, comparing generative vs discriminative
- **Learning Theory**: Bias-variance tradeoff, train vs test error, regularization as a fix for overfitting
- **Deep Learning**: Forward pass, activation functions, loss functions
- **Backpropagation**: Chain rule, computational graphs, gradient flow
- **PCA**: Eigenvalue decomposition, variance maximization, Lagrangian derivation
